# 🍎 Fruit Classifier using Teachable Machine

## Overview
This project is a fruit image classification system developed using Google's Teachable Machine and TensorFlow. The model can identify different types of fruits from uploaded images through a simple Gradio interface.

## Features
- Upload a fruit image.
- Predict the fruit category.
- Display the predicted class with its confidence score.
- User-friendly interface built with Gradio.

## Technologies Used
- Python
- TensorFlow
- Teachable Machine
- Gradio
- NumPy
- Pillow

## Project Files
- `app.py` – Main application.
- `keras_model.h5` – Trained model.
- `labels.txt` – Class labels.

## Dataset
The model was trained using images collected and labeled in Google Teachable Machine.
