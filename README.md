# 🍎 Fruit Classifier using Teachable Machine

## Overview
This project is a fruit image classification system developed using Google's Teachable Machine and TensorFlow. The model classifies fruit images through a simple and interactive Gradio interface.

## Features
- Upload a fruit image.
- Predict the fruit category.
- Display the predicted class and confidence score.
- Simple and user-friendly interface.

## Technologies Used
- Python
- TensorFlow
- Teachable Machine
- Gradio
- NumPy
- Pillow

## Project Files
- `app.py` – Main application.
- `keras_model.h5` – Trained model.
- `labels.txt` – Class labels.

## Dataset
The model was trained using a fruit image dataset from Kaggle.

Dataset:
https://www.kaggle.com/datasets/moltean/fruits
